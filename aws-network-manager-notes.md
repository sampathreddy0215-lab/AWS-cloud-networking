# AWS Network Manager

## What is AWS Network Manager?

AWS Network Manager is a managed service that provides centralized monitoring, visualization, and management of AWS and hybrid cloud networks.

## Benefits

- Centralized network visibility
- Global network monitoring
- Simplified troubleshooting
- Network topology visualization
- Route analysis

## Components

### Global Networks

Logical representation of enterprise networks.

### Transit Gateway Integration

Monitors Transit Gateway attachments and connectivity.

### Cloud WAN Integration

Provides visibility into Core Networks and Segments.

### Direct Connect Integration

Monitors Direct Connect connections and Virtual Interfaces.

### VPN Integration

Displays Site-to-Site VPN tunnel status and health.

### Network Topology

Visual representation of connected AWS resources.

### Route Analysis

Helps verify connectivity between network resources.

## Enterprise Use Cases

- Multi-Region Networking
- Hybrid Cloud Connectivity
- Enterprise Network Operations
- Network Monitoring
- Centralized Visibility

## Best Practices

- Register all network resources
- Monitor VPN tunnel health
- Use topology maps for troubleshooting
- Perform regular route analysis
- Integrate with Cloud WAN and Transit Gateway
