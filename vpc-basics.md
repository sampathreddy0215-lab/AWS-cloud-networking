# VPC Basics

## What is a VPC?
A VPC (Virtual Private Cloud) is a logically isolated network inside AWS.

## CIDR Example
10.0.0.0/16

## Public Subnet
Public subnets allow internet access through an Internet Gateway.

Examples:
- Web servers
- Bastion hosts

## Private Subnet
Private subnets do not have direct internet access.

Examples:
- Databases
- Internal application servers

## Key Components
- VPC
- Subnets
- Route Tables
- Internet Gateway
- Network ACLs
