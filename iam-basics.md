# IAM Basics

## What is IAM?
IAM (Identity and Access Management) helps securely manage access to AWS resources.

## IAM Components
- Users
- Groups
- Policies
- Roles

## Best Practices
- Avoid using root account daily
- Enable MFA
- Use least privilege access
